# 🩸 Community Blood Donor Finder

A React-based web application designed to streamline the process of finding blood donors. This project demonstrates core React concepts including data fetching, state management, and dynamic filtering.

## 🚀 Features

- **Live Data Fetching:** Retrieves donor information from a mock API (JSONPlaceholder) using `useEffect`.
- **Advanced Filtering:** Users can filter donors by specific blood groups (A+, B-, O+, etc.).
- **Real-time Availability:** Displays donor status and provides a visual "Request Help" workflow.
- **Dynamic Statistics:** Automatically updates the total count of available donors based on active filters.
- **Responsive UI:** A clean, grid-based layout that works across different screen sizes.
